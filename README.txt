Title: Scheduling App
Purpose: The purpose of this application is to help a business to create appointments for customers, using a database
to store, manipulate, and synchronize data.

Author: Febechukwu Megwalu
Contact Information: fmegwal@wgu.edu
Student Application Version: 1.0
date: 8/19/2022

IDE: IntelliJ 2021.1.3
JDK version: 17.0.1
JavaFX version: JavaFX-SDK-17.0.1

How to run the program:
- To login, user must put correct username and password.
(Note: This app automatically shows total number of customer appointments
by type and month)
- Immediately after login, this app shows if there is an upcomming
appointment within 15 minutes of login.
- To check total number of contacts and each customer schedules click
the 'Schedule' button on the top-left conner right after login page.
- To filter app by month or week, select corresponding radio button
on appointment column.
- To add, update, and delete customer or appointment, click on their
corresponding buttons.

Additional Report Description:
When user clicks the schedule button, a report of total number of contacts in database is shown on the top of the page.

MySQL Connector Driver: mysql-connector-java-8.0.26
